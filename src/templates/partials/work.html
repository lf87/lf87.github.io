Medical Protection Society

Creation Date: April 2017

This was a fun project to work on as it had to support IE8 (only a small bit of a sarcasm!)., It's fully responsive, and as per the client's request - right down to IE8. Hardware accelerated transitions were used for modern browsers, and JS was used to detect older versions of IE so that they could gracefully fallback to alternative and supported transitions. GSAP was used for the majority of the animations. The seemingly complex folder structure needed so that it could easily be integrated into Sitefinity upon completion.

Gulp Email Boilerplate

Creation Date: March 2017

Because none of the current email boilerplates and build systems available I tried were suitable for working with PSD to HTML email template creation, I decided to create my own. It's loosely based on the Gulp Barebones Boilerplate, but also includes a separate production task that runs a series of other tasks to create a production ready email template. Some of these tasks include inlining of styles, and media query injection
